
# Projeto JAVA Básico (POO - UML)

Projeto desenvolvido para o Bootcamp do Santander de Backend com Java. O Projeto consiste em desnvolver uma parte de um diagrama UML baseado no app do ipod, chamada e internet do lançamento do iphone 2007. 

Funcionalidades a Modelar
- **Reprodutor Musical**
    
    **Métodos:** tocar(), pausar(), selecionarMusica(String musica)

- **Aparelho Telefônico**
    
    **Métodos:** ligar(String numero), atender(), iniciarCorreioVoz()

- **Navegador na Internet**
    
    **Métodos:** exibirPagina(String url), adicionarNovaAba(), atualizarPagina()
